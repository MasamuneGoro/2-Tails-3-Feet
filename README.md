# 2 Tails 3 Feet (Vite + React)

A tiny text-based / low-graphics sandbox-MMO MVP implementing:
- 1 biome (Sticky, Level 1)
- Explore / Find Food / Craft / Recover loops
- Two-tail equipment (2 slots) + one shoe
- Hidden numeric effects for Tail Curler + Chomper (flavor-only)
- Projected Cost Preview for actions

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
